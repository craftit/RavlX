Each module is a unique handle and body class. The handle is derived from the base handle MemModuleC, and the body from the corresponding abstract body base class, MemModuleBodyC. The handle and most of the body are generated automatically when the module is declared in an appropriate header file using the DECLARE_RAVL_MEM_MODULE macro. For example the module "MyModule" is declared thus:

DECLARE_RAVL_MEM_MODULE(MyModule);

This will generate all of class MyModuleC, and all of MyModuleBodyC apart from the application member MyModuleBodyC::Run()
