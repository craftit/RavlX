Requires library RavlImgIO1394dc

Very preliminary support for firewire cameras using the IIDC protocol. Currently only supports the capture of grey scale images.

To use the virtual files mechanism, use @IIDC as the i/p file name. For multiple cameras, use @IIDC:#0, @IIDC:#1 etc.

For a list of cameras supporting this protocol see: http://www.tele.ucl.ac.be/PEOPLE/DOUXCHAMPS/ieee1394/cameras
This driver requires the libdc1394 library which can be found at: http://sourceforge.net/projects/libdc1394
