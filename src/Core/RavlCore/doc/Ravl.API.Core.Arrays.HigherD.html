RAVL provides template classes for 1-D, 2-D and 3-D arrays. If you need a higher-dimensional array, they can be composed from lower-dimensional ones. For example if you need a 5-D 3×4×5×6×7 array of reals, you could create it like this: 

  Array2dC<Array3dC<RealT> > p(3,4);
  for(Array2dIterC<Array3dC<RealT> > i(p); i; ++i) *i = Array3dC<RealT>(5,6,7);
If you need to initialise it with a constant value, this will do it: 
  Array3dC<RealT> tmp(5,6,7);
  tmp.Fill(3.14159);
  Array2dC<Array3dC<RealT> > p(3,4);
  for(Array2dIterC<Array3dC<RealT> > i(p); i; ++i) *i = tmp.Copy();
Individual elements can be accessed in the usual way: 
  cout << p[1][3][3][4][6] << endl;
To iterate through the whole array, you will need to nest the appropriate iterators: 
  for(Array2dIterC<Array3dC<RealT> > i1(p); i1; ++i1)
    for(Array3dIterC<RealT> i2(*i1); i2; ++i2)
      *i2 = j++;
To iterate through 2 (or more) such arrays, it is probably easiest to set up 2 (or more) such nested iterators separately.

The output stream operator will produce a meaningful (if inelegant) output: 

  cout << p << endl;
